# Qt PDF Reader (轻量级 PDF 阅读器)

这是一个基于 Qt 框架开发的轻量级 PDF 阅读器。它支持基本的阅读功能，并具备多文档阅读历史记忆功能。

## 主要功能

* **PDF 渲染**：基于 `Qt PDF` 模块，支持高清展示。
* **阅读历史记忆**：使用 SQLite 数据库存储，自动记录每个文件的最后阅读页码和缩放比例。
* **智能导航**：支持侧边栏目录跳转（书签）和缩略图预览。
* **文本搜索**：支持文档内关键字检索并高亮显示。
* **灵活缩放**：提供“适合宽度”、“适合页面”以及自定义百分比缩放功能。

## 开发环境

* **框架**: Qt 6.x (推荐 6.4+)
* **模块**: `pdf`, `pdfwidgets`, `sql`
* **数据库**: SQLite 3
* **编译器**: MSVC 2019/2022 或 GCC
